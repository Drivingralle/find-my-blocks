# Tag

A `Tag` is used to show tiny bits of important information to users. They should
not consist more then a couple words and when possible have an icon to represent
what the tag represents.

## Usage

```ts
import { Tag } from "../components/Tag";
```

<Playground>
  <Tag variation="default" label="Default tag" />
  <Tag variation="info" icon="info" label="Info tag" />
  <Tag variation="warning" icon="alert-triangle" label="Warning tag" />
  <Tag variation="error" icon="stop-circle" label="Error tag" />
</Playground>
