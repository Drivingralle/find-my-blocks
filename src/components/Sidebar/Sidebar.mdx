# Sidebar

`Sidebar` is a layout component that is used to show a Logo, Filter, and
Navigation. The navigation can be sorted by different values.

## Usage

```ts
import { Sidebar } from "../components/Sidebar";
```

<Playground>
  {() => {
    const [query, setQuery] = useState("");
    const blocks = data.map((item) => ({
      label: item.name,
      count: item.posts.length,
    }));
    return (
      <Sidebar
        blocks={blocks}
        query={query}
        onChange={setQuery}
        onClick={(value) => console.log({ value })}
      />
    );
  }}
</Playground>
