# Link

A very common link. Used to link out to other URLs.

## Usage

```ts
import { Link } from "@fmb/components/Link";
```

<Playground>
  <Link url="https://eddysims.com" external>
    Look at this amazing website!
  </Link>
</Playground>

## Props

<Props of={Link} />

## With Icon

A link can be prepended with an `icon`. A list of available icons can be found
in the [Icon](/components/icon) component.

<Playground>
  <Link url="https://eddysims.com" icon="camera">
    Look at this amazing website!
  </Link>
</Playground>
