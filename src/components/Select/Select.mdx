# Select

`Select` helps users choose a value from a defined list of options.

## Usage

```ts
import { Select, Option } from '../components/Select';
```

<Playground>
    <Select onChange={(val) => alert(`My fav NBA team are the ${val}`)}>
        <Option value="raptors">Toronto</Option>
        <Option value="lakers" selected>Los Angeles</Option>
        <Option value="celtics">Boston</Option>
    </Select>
</Playground>
